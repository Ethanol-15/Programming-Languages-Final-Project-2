Project Setup
Download Spring Boot Project Folder in Google Drive
(https://drive.google.com/drive/u/1/folders/1GQSYy-mIHUcpJydzsCaE_WbPRX23sNJM)

Run the project in IDE (eclipseIDE)
Change directory in terminal to the syntax checker
	
cd C:\Users\???\Downloads\syntaxchecker\syntaxchecker

Run Maven Commands

mvn clean install
mvn spring-boot:run

If an error occurs in running Maven commands, check if you have Maven setup in your computer and run the command mvn -v  in CMD. If Maven is not setup, download it from the provided link (apache-maven-3.9.9-bin.zip)
https://maven.apache.org/download.cgi
